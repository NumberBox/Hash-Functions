Хеширование методом деления


Наиболее простой и часто используемый на практике метод хеширования - метод деления. Он состоит всего из двух шагов. Сперва прообраз должен быть преобразован в целое число, а затем полученное значение вписывается в диапазон 0...n-1 с помощью оператора получения остатка.


Схематичное отображение хеш-функции выглядит следующим образом:


h(x) = x mod M


x - хешируемые строковые данные (числовое представление),

M - количество всевозможных хешей.


Разбор примера:

Сначала строка "Привет мир" была преобразована в числовое представление (1684). Далее по числовому представлению был получен хеш по методу деления (04). Таким образом для входных данных "Привет мир" хеш-значение равно 04.


Задание

По известным данным (исходное сообщение, хеш-код сообщения) найти количество всевозможных выходных хешей (M).


Подсказка:

M лежит в диапазоне от 10 до 25.

Опирайтесь на зависимость между числовым значением строки и полученным хешем.
