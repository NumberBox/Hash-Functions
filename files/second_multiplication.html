Хеширование методом умножения


Метод умножения (мультипликативный метод) представляет из себя вычисление хеш-кода следующей функцией:


h(x) = [M*{(x*A)}]


x - хешируемые данные (числовое представление),

M - количество всевозможных хешей,

A - рациональное число, по модулю меньшее единицы.


Правая часть функции содержит три пары скобок:

1) "()" - скобки приоритета,

2) "{}" - скобки взятия дробной части,

3) "[]" - скобки взятия целой части.


Рациональное число формируется делением величины на две части, при котором отношение большей части к меньшей равно отношению всей величины к ее большей части (А = 0.618033).


Разбор примера:

Сначала строка "Привет мир" была преобразована в числовое представление (1684). Далее числовой код был переемножен с рациональным числом. Потом по формуле были проведены необходимые операции взятия целой и дробной части. В итоге для строки "Привет мир" хеш-значение равно 7.


Задание

Для строки "Золотое сечение" найти хеш-код.


Подсказка:

Количество всевозможных хешей (M) берется из предыдущего пункта.
